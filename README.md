# manygrainsofsalt-code

Code, experiments, and reproduction materials for the [manygrainsofsalt](https://manygrainsofsalt.substack.com/) newsletter.

## Structure

```
posts/
├── bugs-bunny-attack/     # The Bugs Bunny Attack (LLM contradiction reflex probe)
├── <next-post-slug>/
└── ...
```

Each post folder is self-contained with its own README, code, and data.

## Running

Most experiments use Python with `uv` for dependency management. Per-post READMEs have specific instructions.
